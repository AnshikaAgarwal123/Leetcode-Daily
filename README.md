# Leetcode Daily
This repository is a dedicated space for my daily habit of solving LeetCode problems. As a CSE student, my goal is to build a strong foundation in Data Structures and Algorithms (DSA) using Java.

I believe that consistency is the key to mastering logic, and this repo tracks my progress, one problem at a time.

🧠 My 3-Step Process
For every challenge, I follow a simple workflow to ensure I’m actually learning, not just coding:
Understand: Break down the problem constraints and edge cases.
Solve: Start with a working "Brute Force" logic.
Optimize: Refine the code to improve Time and Space complexity.
